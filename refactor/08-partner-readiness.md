# Fase 8 — Partner readiness

## Objetivo
Deixar o Midgard pronto para ser recipiente oficial no futuro.

## Entregas
- Criar `Partner` (ou ao menos deixar `originType = partner` utilizável)
- Suportar sources `official` (Midgard) e `partner` como “oficiais”
- Permitir que `Manga.preferredSourceId` aponte para uma source oficial do Midgard ou do parceiro
- Preparar fluxo para ingestão oficial sem depender do provider externo atual
